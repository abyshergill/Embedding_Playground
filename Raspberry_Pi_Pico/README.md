# Raspberry Pi Pico Project

![Raspberry Pi Pico](assest/main.jpg)

## Overview
The **Raspberry Pi Pico** is a low-cost, high-performance microcontroller board built around the **RP2040 chip**. It is designed for small projects, prototyping, and embedded systems, offering flexibility and ease of use for hobbyists and developers.

## Key Features
- Dual-core ARM Cortex-M0+ processor
- 264KB of SRAM
- 2MB of onboard Flash memory
- USB 1.1 device and host support
- 26 multi-function GPIO pins
- Support for C/C++ and MicroPython programming

## Why Use Pico?
- Affordable and beginner-friendly
- Compact size for small projects
- Strong community support
- Easy integration with sensors, actuators, and other peripherals
